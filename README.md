# ClaudeBridge

Управляй сессиями **Claude Code в VS Code прямо с телефона** — смотри чаты в реальном времени, пиши промпты текстом или голосом, отвечай на подтверждения, переключай режим/модель, получай пуш-уведомления. Работает удалённо, из любой точки, без второго приложения на телефоне.

Каждый запускает программу у себя на **своём ПК со своей подпиской Claude**. Мост ничего не берёт из аккаунта — он просто зеркалит окно VS Code на телефон.

---

## Что нужно
- **VS Code** с расширением **Claude Code** (установлено и вы залогинены в Claude).
- Компьютер на **Windows** или **macOS**.
- Телефон с браузером (Chrome на Android / Safari на iPhone).

## Установка (2 минуты)

### 1. Скачай программу
- **Windows** → [**ClaudeBridge-windows.exe**](https://github.com/Junfey/ClaudeBridge/releases/latest/download/ClaudeBridge-windows.exe)
- **macOS** → [**ClaudeBridge-macos**](https://github.com/Junfey/ClaudeBridge/releases/latest/download/ClaudeBridge-macos)

Или зайди на страницу [**всех релизов**](https://github.com/Junfey/ClaudeBridge/releases/latest) и скачай нужный файл.

### 2. Запусти
- **Windows**: двойной клик по `ClaudeBridge-windows.exe`.
  Если появится «Система Windows защитила ваш компьютер» — нажми **«Подробнее» → «Выполнить в любом случае»** (программа не подписана платным сертификатом — это нормально для такого инструмента).
- **macOS**: в Терминале выполни `chmod +x ~/Downloads/ClaudeBridge-macos`, затем в Finder **правой кнопкой по файлу → Open** (первый раз — иначе macOS блокирует неподписанное).

### 3. Подключи телефон
- Откроется окно с **QR-кодом** и ссылкой.
- Если внизу окна красным написано «VS Code без отладки — нажми, чтобы перезапустить» — **нажми** (сохрани файлы заранее: VS Code перезапустится).
- На телефоне **наведи камеру на QR** или скопируй ссылку кнопкой «Копировать».
- (рекомендуется) в браузере на телефоне: меню → **«Установить приложение» / «На экран Домой»** — иконка появится как у обычного приложения.

Готово — вкладки Claude из VS Code появятся на телефоне.

## Нюансы
- **Первый заход**: сервис-туннель (localtunnel) покажет страницу-предупреждение и попросит ввести **IP-адрес** — он написан прямо в окне ClaudeBridge на ПК. Введи один раз, браузер запомнит.
- **Держи окно ClaudeBridge открытым** на ПК (можно свернуть) — это и есть мост. Хочешь, чтобы поднималось само после перезагрузки ПК — поставь галку **«Запускать при включении ПК»**.
- **Пуш-уведомления** включены по умолчанию: придёт уведомление, когда Claude ответит во вкладке, которую ты сейчас не смотришь (даже если приложение закрыто). Нажатие по пушу открывает нужный чат.

## Обновления
Программа сама проверяет новые версии на GitHub и показывает кнопку **«🔄 Обновить»** — жмёшь, она скачивает обновление и перезапускается. Ничего вручную качать не нужно.

## Что внутри
Мост на FastAPI + PWA + Chrome DevTools Protocol к вебвью Claude в VS Code, чтение канонических JSONL-файлов Claude для точной истории, пуш через Web Push, туннель на чистом Python (без Node). Один самодостаточный файл.
